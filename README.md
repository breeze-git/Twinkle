Twinkle — очень простой и незамысловатый таймер, разработанный на языке Python для моих личных нужд и просто попрактиковаться в написании кода.

Если все же кто-то надумает это использовать, то вся информация по управлению этим таймером, которая вам может понадобится, есть в подсказках по нажатию на клавишу h в приложении.
